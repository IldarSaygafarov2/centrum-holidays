{% trans 'Электронная туристическая виза' %}

{% blocktrans %}Резиденты вышеперечисленных государств также имеют возможность получить электронную туристическую визу, заполнив анкету на портале E-VISA и оплатив пошлину в размере 2 0 USD. В случае замены паспорта в период действия электронной визы, электронную визу необходимо получить повторно.{% endblocktrans %}

{% trans 'Въезд с приглашением' %}

{% blocktrans %}Для граждан всех остальных стран въездная виза оформляется на основании визовой поддержки. Для оформления визовой поддержки, Вам необходимо предоставить нашей компании следующий список документов:{% endblocktrans %}

    {% blocktrans %}
  • Заполненную анкету для визовой поддержки (скачать)
  • Сканированную копию Вашего паспорта
  • Сканированную копию справки с места работы
    • {% endblocktrans %}

{% blocktrans %}После получения нами всей необходимой информации, мы оформим визовую поддержку в МИД Республики Узбекистан в Ташкенте которое рассмотрит Ваши документы в течение 7 рабочих дней.{% endblocktrans %}

{% blocktrans %} МИД Республики Узбекистан оставляет имеет право отказать в выдаче разрешения, не предоставляя причин отказа. В случае отказа уплаченная пошлина не возвращается. Получив разрешение МИД, мы вышлем Вам на почту номер телекса после чего Вы сможете обратиться в дипломатическое представительство или консульство Республики Узбекистан в стране вашего пребывания с перечнем следующих документов:{% endblocktrans %}

    {% blocktrans %}
  • Оригинал паспорта (срок действия которого должен превышать срок действия визы не менее чем на 3 месяца);
  • Заполненная визовая анкета в 2 экземплярах;
  • 2 цветные фотографии (паспортного размера);
  • Номер телекса (высланный Вам на почту);
  • Консульский сбор (согласно тарифам приведенным в разделе Пошлина)
    • {% endblocktrans %}

{% trans 'Длительность рассмотрения поданных документов составляет 3 рабочих дня.' %}

  • Сканированную копию справки с места работы
  • Сканированные копии авиабилетов или брони
    • {% endblocktrans %}

{% blocktrans %}После получения нами всей необходимой информации, мы оформим визовую поддержку в МИД Республики Узбекистан в Ташкенте которое рассмотрит Ваши документы в течение 7 рабочих дней.{% endblocktrans %}

{% blocktrans %}Получив разрешение МИД, мы вышлем Вам на почту официальный разрешающий документ с номером телекса.Данный документ должен быть предъявлен Вами при прохождении паспортного контроля в стране Вашего пребывания и на его основании Вам оформляется виза по прибытию в международный аэропорт имени «Ислама Каримова» в секторе визового обеспечения МИД Республики Узбекистан.{% endblocktrans %}
